# Matrice di Mapping

La **matrice di mapping** è uno strumento utilizzato nella progettazione software per mettere in relazione i **requisiti funzionali (RF)** con i **casi d’uso (UC)**.

## Legenda

- **RF (Requisiti Funzionali):** descrivono cosa deve fare il sistema  
- **UC (Casi d’Uso):** descrivono come gli utenti interagiscono con il sistema
